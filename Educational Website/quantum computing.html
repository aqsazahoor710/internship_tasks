Quantum Computing Developer – 12-Week Roadmap

Weekly Learning Plan

Goals: Understand classical vs quantum computing, qubits, superposition, entanglement.

IBM Quantum Fundamentals

Project: Install Qiskit & Python; simulate a simple qubit superposition.

Goals: Learn complex numbers, linear algebra basics, vectors.

Linear Algebra Refresher (Khan Academy)

Project: Practice bra-ket notation exercises.

Goals: Hermitian/unitary matrices, tensor products, inner products.

Quantum Mechanics Math (YouTube)

Project: Create simple matrix operations using Python (NumPy).

Goals: Pauli-X/Y/Z, Hadamard, CNOT, phase gates; learn gate composition.

Qiskit: Quantum Gates

Project: Build quantum circuits with 1-2 qubits in Qiskit.

Goals: Circuit diagrams, qubit measurement, probability amplitudes.

Qiskit Tutorials

Project: Measure qubits and visualize the results.

Goals: Deutsch-Jozsa algorithm, Grover’s search algorithm.

Grover’s Algorithm Tutorial

Project: Implement Deutsch-Jozsa algorithm on a simulator.

Goals: Shor’s factoring, Quantum Fourier Transform (QFT).

Shor’s Algorithm Lecture Notes

Project: Simulate small-number factoring in Qiskit.

Goals: Programming quantum circuits using Qiskit & Cirq SDKs.

Qiskit Documentation

Project: Build multi-gate circuits programmatically in Python.

Goals: Learn about qubit types, decoherence, error correction.

IBM Quantum Experience

Project: Run a small circuit on IBM Quantum real device.

Goals: Optimization problems, cryptography, post-quantum security.

MyTectra Quantum Training

Project: Solve a small optimization problem using VQE or Grover’s algorithm.

Goals: Quantum Machine Learning, finance, chemistry.

Quantum ML Tutorial (YouTube)

Project: Apply a quantum ML library to a small dataset.

Goals: Integrate knowledge to build a small quantum application.

Project Examples: Quantum search algorithm, random number generator, or small dApp on IBM Quantum real device.
